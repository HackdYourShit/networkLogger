CoreLocationHelper.swift

CoreLocation Helper is a reusable class which is actually a wrapper around CoreLocation.framework features. Say, You want to work on Location based service then, you can use this code to implement core functionality without any hazel. It's based on Closure Pattern and Observer Pattern (i.e. Notifications). Along with this, I've coved Geocoding and Reverse Geocoding.
